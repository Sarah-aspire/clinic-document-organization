# Validation Rules

This document defines the basic validation rules used in the clinic document organization project.

## Data Quality Checks

| Check | Purpose |
|---|---|
| Missing status check | Identify records with missing or incomplete document status |
| Duplicate status check | Detect records marked as duplicated |
| Pending records check | Highlight documents that still require action |
| Department overview | Understand which departments generate the most administrative workload |
| Responsible staff overview | Identify ownership and follow-up responsibilities |

## Business Purpose

The validation rules help improve document transparency, reduce manual searching effort and support better follow-up in daily clinic administration.
