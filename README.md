# Clinic Document Organization

A practical data and document organization project for a small clinic scenario.

## Project Overview

This project demonstrates how scattered Excel files, administrative records and operational documents can be organized into a clearer, more reliable and easier-to-use workflow structure.

The scenario is based on a small clinic environment where documents, appointment lists, invoices and administrative data were stored inconsistently across different files and formats.

## Project Goals

- Organize scattered administrative data
- Improve file and document structure
- Clean inconsistent Excel records
- Detect missing or duplicate information
- Create a clearer workflow overview
- Support better reporting and daily operations

## Tools

- Excel
- Python / Pandas
- Power BI
- Data quality checks
- Process documentation
